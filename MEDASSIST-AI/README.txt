MEDASSIST AI - AI HEALTH ASSISTANT WEB APPLICATION

Introduction
  MedAssist AI is an intelligent health assessment web application developed with the goal of applying AI technology to community healthcare. The project aligns with the "AI for Humanity" philosophy of Trivita AI.

Key Features
1. BMI calculation with visual charts
2. Heart rate assessment based on age and activity level
3. AI chatbot for health symptom consultation
4. Assessment history storage (LocalStorage)
5. Dark/Light mode toggle
6. Voice recognition (Vietnamese language support)
7. Fully responsive across all devices
8. Modern, user-friendly interface

Technologies Used
- HTML5, CSS3, JavaScript
- LocalStorage API
- Web Speech API (voice recognition)
- Responsive Web Design
- Font Awesome Icons
- Google Fonts (Inter)

Getting Started

LIVE DEMO:
  https://khanh3-dat3.github.io/MEDASSIST-AI

### Alternative Method
Simply download the project files and open `index.html` directly in any modern web browser.

Customization
- **Change logo:** Modify the `.logo` section in HTML/CSS
- **Change colors:** Adjust CSS variables in the `:root` selector
- **Add features:** Extend functionality in `script.js`

Important Notes
- The AI chatbot uses rule-based logic, not actual machine learning AI
- All data is stored locally in the browser (LocalStorage)
- This is a support tool and does not replace professional medical diagnosis
- Voice recognition only works on Chrome/Edge browsers
- For educational and demonstration purposes only
